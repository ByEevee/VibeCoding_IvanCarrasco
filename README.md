# 🟣 Pokémon Corona

**Pokémon Corona** es un videojuego fan-made desarrollado con RPG Maker XP, donde el jugador se embarca en una aventura inspirada en la saga Pokémon.

> ⚠️ El juego aún está en desarrollo.

---

## 🎮 Descripción del juego

En **Pokémon Corona**, el jugador asume el rol de un entrenador que inicia su viaje en una región original.

Actualmente el juego incluye:

- Exploración básica 🗺️  
- Sistema inicial de combate ⚔️  
- Interacción con el entorno 👤  

Funcionalidades previstas:

- Sistema completo de combate por turnos  
- Captura de Pokémon  
- Progresión del jugador  
- Historia principal  

---

## 🧠 Objetivo del proyecto

Este proyecto forma parte de la asignatura **Entornos de Desarrollo**.

El objetivo es:

- Aplicar el proceso completo de desarrollo de software  
- Diseñar antes de programar  
- Verificar y depurar el sistema  
- Documentar todas las fases del desarrollo  

---
